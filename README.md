# Yummy 🍲
### A dynamic Food Recipes Web Application built with JavaScript, Fetch API, and responsive UI. 

Explore global cuisines, ingredients, and step-by-step cooking instructions. This project was developed as a professional Front-end exam to demonstrate modern web development skills.

## 🚀 المميزات (Key Features)
* **Live Search:** بحث لحظي عن الأكلات بالاسم أو الحرف الأول.
* **Global Cuisines:** تصفح المأكولات حسب الدولة (Area) أو التصنيف (Category).
* **Recipe Details:** عرض المكونات (Ingredients) وطريقة التحضير (Instructions) بشكل منظم.
* **Video Tutorials:** روابط مباشرة لليوتيوب لكل وصفة.
* **Fully Responsive:** متوافق تماماً مع جميع الشاشات (Mobile, Tablet, Desktop).

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **Languages:** HTML5, CSS3, JavaScript (ES6+).
* **Libraries:** jQuery, Bootstrap.
* **APIs:** TheMealDB API.
* **Deployment:** GitHub Pages.
